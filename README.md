# 📝 Python To-Do CLI App

## 📌 Description
A simple command-line To-Do application built using Python.  
Users can add, view, and delete tasks.

## 🚀 Features
- Add tasks
- View tasks with index
- Delete tasks safely
- Input validation

## 🧠 Concepts Used
- Loops (while, for)
- Conditions (if-elif-else)
- Lists
- Functions

## ▶️ How to Run

1. Make sure Python is installed  
2. Run the file:

```bash
python project1_todo_cli.py
